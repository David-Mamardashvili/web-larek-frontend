# Проектная работа "Веб-ларек"

Стек: HTML, SCSS, TS, Webpack

Структура проекта:
- src/ — исходные файлы проекта
- src/components/ — папка с JS компонентами
- src/components/base/ — папка с базовым кодом

Важные файлы:
- src/pages/index.html — HTML-файл главной страницы
- src/types/index.ts — файл с типами
- src/index.ts — точка входа приложения
- src/styles/styles.scss — корневой файл стилей
- src/utils/constants.ts — файл с константами
- src/utils/utils.ts — файл с утилитами

## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

или

```
yarn
yarn start
```
## Сборка

```
npm run build
```

или

```
yarn build
```
## Архитектура проекта

### Model
1) Класс ProductList
Методы: 
-CreateProduct(Берет данные товаров с сервера и вставляет их в разметку с помощью template.)
-Submit(Отправляет данные о товарах, способе оплаты, адресе, почте и телефоне пользователя.)

### View 
1) Класс Modal
Методы:
-OpenModal(Открывает модальное окно продукта, на который было произведено нажатие.)
-CloseModal(Закрывает выбранное модальное окно по нажатию на кнопку "Крестик" или на зону вне модального окна.)

2) Класс Basket
Методы:
-AddProduct(Добавляет выбранный пользователем товар по нажатию на кнопку "Купить")
-RemoveProduct(Удаляет выбранный пользвотелем товар по нажатию на кнопку "Корзина".)
-ClearBasket(Полностью очищает корзину.)
-BasketTotalAmount(Суммирует стоимость всех товаров в корзине.)

3) Класс Payment 
Методы:
-PaymentMethod(Переключает способ оплаты по нажатию.)
-AddressError(Выводит ошибку, если адрес не введён и удерживает кнопку неактивной, пока адрес не введен.)

4) Класс Contacts 
Методы:
-ContactsError(Выводит ошибку, если одно из полей не введено и удерживает кнопку неактивной, пока оба поля не введены.)

5) Класс Result 
Методы:
-CloseButton(Закрывает модальное окно по нажатию на кнопку "За новыми покупками!")
